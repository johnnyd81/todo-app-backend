# Todo-App

# App Description

This app allows a user to create a collection of todos. By entering a todo in the input field and clicking the add todo button the new todo will be added onscreen and be saved in the database. Each user's todos are only accessible by that specific user i.e. adding new todos and deleting the todos. Before using the app a user has to sign up to be authenticated to use the app. Thereafter the user will be allowed to login into the app.

# Running the App

1. In the command terminal from the root directory cd into the to-do-app directory and type "npm install".
2. Next cd into the backend directory and type "npm install".
3. From the backend type "npm run dev" to start the frontend and the backend simultaneously. The backend also connects to the MongoDb database.
